# Geometry Screener

An online interactive geometry readiness screener designed to assess student understanding of foundational geometry concepts through immediate feedback and adaptive-style questioning.

## Planned Features
- Interactive geometry questions
- Immediate feedback
- Student-friendly UI
- Growth tracking
- Browser-based deployment
- Future ScholarForge integration
